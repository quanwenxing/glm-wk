# デザインシステム

## カラーパターン

子供向けのポップでカラフルなパレット。各科目ごとにイメージカラーを設定。

### メインカラー

| カラー名 | HEX | 用途 |
|----------|-----|------|
| Primary Blue | `#3B82F6` | メインボタン、リンク |
| Primary Dark | `#1E40AF` | ホバー状態 |
| Secondary Yellow | `#FBBF24` | アクセント、ハイライト |
| Accent Orange | `#F97316` | 重要な注意喚起 |
| Accent Pink | `#EC4899` | 楽しい装飾 |

### 科目別カラー

| 科目 | カラー名 | HEX | 用途 |
|------|----------|-----|------|
| 国語 | Kokugo Red | `#EF4444` | 国語関連のカード、アイコン |
| 算数 | Sansu Blue | `#3B82F6` | 算数関連のカード、アイコン |
| 理科 | Rika Green | `#22C55E` | 理科関連のカード、アイコン |
| 社会 | Shakai Orange | `#F97316` | 社会関連のカード、アイコン |

### ニュートラルカラー

| カラー名 | HEX | 用途 |
|----------|-----|------|
| Background | `#FEFCE8` | 背景色（クリーム色） |
| Surface | `#FFFFFF` | カード、コンテナ |
| Text Primary | `#1F2937` | メインテキスト |
| Text Secondary | `#6B7280` | 補足テキスト |
| Border | `#E5E7EB` | 枠線 |
| Success | `#22C55E` | 成功、正解 |
| Error | `#EF4444` | エラー、不正解 |

## タイポグラフィ

### フォントファミリー

- **日本語**: Noto Sans JP（Google Fonts）
- **英数字**: Inter（または Geist Sans）

### フォントサイズ

| ラベル | サイズ | 用途 |
|--------|--------|------|
| Display | `text-5xl` (48px) | ヒーローセクションのタイトル |
| H1 | `text-4xl` (36px) | ページタイトル |
| H2 | `text-3xl` (30px) | セクションタイトル |
| H3 | `text-2xl` (24px) | サブセクションタイトル |
| Body Large | `text-lg` (18px) | 本文（強調） |
| Body | `text-base` (16px) | 本文 |
| Body Small | `text-sm` (14px) | 補足情報 |
| Caption | `text-xs` (12px) | 注釈 |

### フォントウェイト

| ラベル | ウェイト | 用途 |
|--------|----------|------|
| Regular | 400 | 通常のテキスト |
| Medium | 500 | 見出し、強調 |
| Semibold | 600 | ボタン、リンク |
| Bold | 700 | 大見出し |

## 間隔（Spacing）

8pxベースのスケールを使用。

| トークン | 値 | 用途 |
|----------|-----|------|
| xs | 4px | 小さい隙間 |
| sm | 8px | 要素間の基本的な隙間 |
| md | 16px | セクション内の余白 |
| lg | 24px | コンポーネント間の余白 |
| xl | 32px | セクション間の余白 |
| 2xl | 48px | 大きな余白 |

## 角丸（Border Radius）

| トークン | 値 | 用途 |
|----------|-----|------|
| sm | 4px | 小さいコンポーネント |
| md | 8px | ボタン、カード |
| lg | 12px | 大きなカード |
| xl | 16px | 特別なセクション |
| full | 9999px | 丸いボタン、バッジ |

## シャドウ

| トークン | 用途 |
|----------|------|
| sm | 小さいカード |
| md | 通常のカード |
| lg | 大きなカード、モーダル |
| xl | 特別な要素 |

## コンポーネントスタイル

### ボタン

- **サイズ**: sm, md, lg
- **バリアント**: primary, secondary, outline, ghost
- **角丸**: md (8px)

### カード

- **背景**: White
- **角丸**: lg (12px)
- **シャドウ**: md
- **ホバー**: 少し浮き上がるアニメーション

### 入力フォーム

- **角丸**: md (8px)
- **フォーカス**: Primary Blueのアウトライン
- **エラー**: Error色のアウトライン

## アニメーション

| 種類 | 時間 | イージング |
|------|------|------------|
| Fast | 150ms | ease-out |
| Base | 200ms | ease-in-out |
| Slow | 300ms | ease-in-out |
